Ca input trebuie setat numarul de intrari n (de preferat 8),
apoi numarul de conexiuni ( <= n) si apoi cele n inputuri
si outputuri grupate (input,output). La fiecare conexiune
spune la fiecare nivel daca comutatorul este in comutare directa
sau inversa. Daca introducem un input si un output si nu se afiseaza
nimic, inseamna ca intre acele doua legaturi nu se mai poate stabili
o conexiunea folosind doar comutatoare directe sau inverse.
